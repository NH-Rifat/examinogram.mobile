# Examinogram Mobile

React Native / Expo mobile app for the Examinogram platform.

## Stack

- Expo SDK ~54
- React 19 + React Compiler
- expo-router (file-based routing)
- TypeScript (strict)

## Getting started

```bash
yarn install
yarn start
```

## Project structure

```
app/          # Screens and layouts (expo-router)
components/   # Shared UI components
constants/    # Theme colors and fonts
hooks/        # Custom React hooks
assets/       # Images and icons
```
